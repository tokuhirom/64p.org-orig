はたして、どうするのがベストなのかはよくしらんけど……。

データ的なのは share/ というルール、かな?

dotfiles は share/dotfiles から soft symbolic link はる
share/dotfiles/
share/dotfiles/.emacs
share/dotfiles/.emacs.d/

howm のデータは
share/howm/

code は dev/ におく。
>||
    dev/foo/
||<

この際、App::gh をつかえば以下のようにすることにより、容易にすべての github にはいっているものを clone/update することが可能
>||
gh all tokuhirom --into ~/dev/
||<
